# This directory contains all of the eqbm metric data/plots 

# Current metrics include:
# Ens Mean (1st central moment)
# Ens Std Dev (2nd central moment)
# Ens Skew (3rd central moment)
# Ens Kurtosis (4th central moment)

# When all four metrics have stabilized with time, we say
# that the ens has reached an eqbm distribution, representing
# distribution of the possible 'physical' states the model
# can achieve with its given inputs/parameters

# For F not constant in time, there should never be a true 
# eqbm distribution
